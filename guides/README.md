## 🚀 **QUICK START (All Languages)**

### Universal Commands:
```bash
# Clone framework
git clone https://github.com/neoand/neodoo18framework.git
cd neodoo18framework

# Create project
./quick-start.sh my_project

# Validate
python3 framework/validator.py my_project/
```
